# GcodeBlasting
带简单的图形码登录框暴力破解工具，图形码再也不能阻挡我的百万字典了。
为某次被图形码阻挡道路时自写的简单爆破工具，简单GUI，能用就行，可能复杂验证时会不可用。

python编写,可自行编译成exe方便使用
